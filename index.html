카드 길이가 변경될 때 나머지 카드 위치와 Hook 키프레임 등 하위 컬렉션 데이터가 함께 이동하도록 Shift 기능을 보강했습니다.

수정
03

04

Drone 줄기와 add2 줄기 작업 시 스위치 키프레임이 섞이던 문제 수정

수정

Drone 줄기와 add2 줄기를 번갈아 작업할 때 DroneMeshObject의 스위치 키프레임이 서로 섞이던 문제를 해결했습니다.

05

RGB 비율 설정을 Export 단계로 이동

개선

기존 RGB 비율 0.7 노드를 삭제하고, Export 단계에서 비율을 설정하도록 변경했습니다. 기본값은 70%입니다.

06

이동 구간에서도 일관되도록 Twinkle 노드 개선

개선

드론이 움직이는 구간에서 Twinkle 효과가 과하게 반짝이던 문제를 개선해 정지·이동 상태 모두에서 일관되게 표시됩니다.

Geometry Nodes에서 Twinkle을 검색한 뒤, Group에 있는 Twinkle을 사용해야 합니다. Coloring의 Twinkle은 Asset Library 노드이므로 동일하게 동작하지 않습니다.

향후 Asset Library 없이 XneoPicasso에서 효과를 직접 선택할 수 있도록 개선할 예정입니다.

25

장면 교체 시 노드 연결 누락 현상 수정

수정

장면을 교체할 때 일부 노드 연결이 누락되던 문제를 수정했습니다.

28

줄기별 시작 Offset 기능 추가

신규

Drone 줄기마다 시작 지연값을 frame 또는 초 단위로 설정할 수 있습니다. Offset을 적용하면 카드 길이는 유지한 채 해당 줄기 전체가 뒤로 밀립니다.

※ 참고불꽃신호 최적화를 이미 완료한 상태에서 시작 Offset을 새로 적용하거나 변경했다면, 변경된 타이밍을 반영하기 위해 불꽃신호 최적화를 다시 실행하는 것을 권장합니다.
29

Offset 적용 시 타임라인·Export 동기화 개선

개선

시작 Offset이 적용된 줄기의 Switch 키프레임, Hook 키프레임, 타임라인 범위가 함께 갱신되도록 개선했습니다. Export 시에는 Offset이 적용된 구간을 반영하되, 출력 파일은 기존처럼 0프레임 기준으로 정리됩니다.

30

카드 빠른 재선택 시 선택 상태 꼬임 방지

수정

애니메이션 편집 모드 전환 중 다른 카드를 빠르게 선택했을 때, 이전 카드의 처리 결과가 뒤늦게 적용되던 문제를 방지했습니다.

31

애니메이션 편집 진입 시 Blender Timeline·Outliner 자동 정렬

개선

Scene 카드의 애니메이션 편집에 들어가면 Blender Timeline이 해당 카드 범위로 자동 정렬되고, Outliner도 활성 오브젝트 위치로 이동합니다.

02

Store

새로운 Store 서버와 안전한 Import 흐름을 제공합니다.

07

별도 장면 Store 구축

신규

기존 Store 장면 업로드의 권한 문제를 해결하기 위해 회사 계정으로 새로운 서버를 구축했습니다. 장면은 Store(test)를 통해 업로드할 수 있습니다. Store(test)의 Upload Scene으로 장면을 등록하고, Hook 키프레임이 포함된 애니메이션은 장면 카드의 path3 업로드로 등록할 수 있습니다.

Local Scene Upload는 업로드 없이 빠르게 Formation 작업을 진행할 때 사용할 수 있습니다.

처음 한 번 로그인해야 하며, 이후 로그인 상태가 유지됩니다. @pabloair.com 계정만 사용할 수 있습니다. 한 달 동안 문제가 없으면 기존 Store 탭을 삭제하고 Store(test)의 이름을 변경할 예정입니다.

※ 참고기존 Store에 등록된 장면의 최대 RGB 값은 0.7입니다. 이를 다시 70%로 Export하면 RGB 값이 0.5 이하로 낮아질 수 있습니다.

22

이륙 대형 Coloring 작업 간소화

개선

Drone 카드에서 이륙 대형 편집을 누르면 TakeOff_Formation_Coloring Collection과 Mesh가 생성되고 Color Manager가 자동 설정됩니다. 이륙 대형의 Color, Animation, 지상 배치 변경은 모두 TakeOff_Formation_Coloring Mesh에서 진행합니다.

지상 배치 변경 후 이륙 대형 동기화를 누르면 DroneMeshObject와 DroneMeshObject_animation이 같은 모양으로 동기화 됩니다.

이륙 대형 Animation은 Hook을 직접 생성하지 않고, TakeOff_Formation_Coloring에서 Vertex를 선택한 뒤 Hook 동기화 기능을 사용하면

두 Mesh에 동일한 Hook이 연결·생성됩니다.

23

장면 카드 이름 식별 기능 추가

신규

장면 카드 이름 영역을 선택해 이름을 입력한 뒤 Enter로 편집할 수 있습니다. 편집한 이름은 Blender에 생성되는 분홍색 Collection에서 식별할 수 있습니다.

Blender 이름 자체를 변경하면 수정 범위가 커지기 때문에 현재는 Collection 식별 방식으로 제공합니다.

24

PCA Coloring을 XneoPicasso에 병합

신규

27

불꽃 신호 부분·전체 삭제 기능 추가

신규

버텍스 선택 삭제는 선택한 버텍스에 적용된 불꽃 신호만 삭제합니다. 전체 삭제는 해당 카드에 적용된 모든 불꽃 신호를 일괄 삭제합니다.

※ 참고불꽃 신호 최적화 이후 불꽃을 수정한 경우에는 불꽃 신호 최적화를 다시 수행해야 합니다.
